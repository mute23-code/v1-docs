# 黑名单

<p><span class="span-bot-admin">Bot 管理员</span> <span class="span-bot-helper">Bot 协助者</span></p>

## 拉黑
```
.屏蔽用户 id id1 id2
```
```
.屏蔽群 id id1 id2
```

## 解禁
```
.解封用户 id id1 id2
```
```
.解封群 id id1 id2
```

## 查看
```
.查看用户黑名单
```
```
.查看群聊黑名单
```

## 快捷方式
群内发送 `.静默` 可快捷屏蔽当前群聊  
群内发送 `.响应` 可快捷解封当前群聊
