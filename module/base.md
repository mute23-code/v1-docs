# 基础模块
<p><span class="span-friend">私聊</span> <span class="span-group">群聊</span></p>

林汐 最基础的运行模块

----

## Ping
---
测试 林汐 是否在线
```
.ping
```

## 复读
---
当群内相同消息 ≥ 2 时，林汐会进行一次复读

## 帮助
---
获取林汐官方群群号、文档链接、反馈平台
```
<at> 林汐
```
> 或直接输入`林汐`

## 签到
---
获取金币的一种方式
```
.签到
```

## 金币
---
可以使用[签到](./module/base?id=签到)获取金币

### 查询
查询当前账号的硬币数量
```
.我的金币
```

### 转账
将当前账号的金币转让给其他人。
```
.转账 <at> <金币数量>
```

### 金币排行
```
.金币排行
```
