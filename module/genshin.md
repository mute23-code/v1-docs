## 角色面板

### 查看面板（未绑定UID）：
查看该UID角色展柜的所有角色
```
.原神面板 <UID>
```
查看该UID的角色面板
```
.原神面板 <角色> <UID>
```


### 查看面板（已绑定UID）：
```
.原神绑定 <UID> (绑定UID至发送此指令的QQ；QQ已绑定则会更新绑定的UID)
```
```
.原神面板 [角色] [at]（查看at的人的角色面板；默认自己）
```

