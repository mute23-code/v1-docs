<p><span class="span-friend">私聊</span>  <span class="span-group">群聊</span></p>

提供魔方的相关功能

----
## 打乱
发送魔方打乱公式
目前支持打乱：
* 2-7阶
* 魔表（clock）| 金字塔（pyram）| 斜转（skewb）| sq1（sq1）
```
.<阶数>
```

<details>
<summary>指令别名</summary>
<pre><code>
.2 = .二阶打乱
.3 = .三阶打乱
.4 = .四阶打乱
.5 = .五阶打乱
.6 = .六阶打乱
.7 = .七阶打乱
.cl = .clock = .魔表打乱
.pyram = .pyraminx = .金字塔打乱
.skewb = .斜转打乱
.sq1 = .square = .square-1 = .sq1打乱
</code></pre>
</details>

## 公式
发送部分魔方公式  
目前支持公式：  
* 三阶：cfop
* 四阶：o特、p特

```
.cfop
```
```
.o特|.p特|.四阶
```

## 视频教程
发送b站视频教程  
目前支持教程：
* 二阶：二阶二步法
* 三阶：zz法 | op法
* 四阶：yau法
* 五阶：84法

    输入 `.目录` 可查询命令
