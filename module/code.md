<p><span class="span-friend">私聊</span> <span class="span-group">群聊</span></p>

## 指令
```
.code <语言> [-i] [inputText]
<代码>
```
> `-i`：输入，后跟输入内容

## 实例
无输入
```py
.code py
    print("测试")
```

有输入
```py
.code py -i 10
a = input()
for i in range(int(a)):
    print(i)
```
