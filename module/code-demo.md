# 代码示例


## Python 3
```py
print("Hello, World!")
```

## Node.js
```js
console.log("Hello, World!")
```

## C
```c
#include <stdio.h>
int main()
{
    printf("Hello, World! \n");
    return 0;
}
```

## C#
```c#
using System;
namespace HelloWorldApplication
{
    class HelloWorld
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!");
            Console.ReadKey();
        }
    }
}
```

## C++
```cpp
#include <iostream>
using namespace std;
int main()
{
    cout << "Hello, world!" << endl;
    return 0;
}
```

## Java
```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
> **注意**  
> 类名必须为 **Main**。

## PHP
```php
<?php
echo "Hello, World!";
?>
```

## Golang
```go
package main;
import "fmt";
func main() {
    fmt.Println("Hello, World!");
}
```
