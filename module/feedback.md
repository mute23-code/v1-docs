# 广播
---

## 反馈广播
<p><span class="span-group">群聊</span></p>

将内容发送给超管
```
.send <内容>
```

> 当然你也可以通过 [反馈平台](https://support.qq.com/product/423251) 进行反馈

## 通知广播
<p><span class="span-bot-admin">Bot 管理员</span>  <span class="span-bot-helper">Bot 协助者</span></p>

仅限超管私聊，林汐会向所有加入的群聊广播内容，每次发送有5sCD。
```
.notice <内容>
```
