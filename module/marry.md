# 娶群友
<p><span class="span-group">群聊</span></p>

## 功能介绍

纯爱 双向奔赴版，每天刷新一次，两个人会互相抽到对方
```
.娶群友
```

ntr 狂喜版，每次抽到的结果都不一样。
```
.透群友
```

离婚  
~~雪花飘飘北风萧萧，天地一片苍茫~~
```
.离婚
```

查看本群cp
```
.本群cp
```

查看当前群可以娶到的群友列表
```
.群友卡池
```

查看被透记录（跨群）
```
.涩涩记录
```
